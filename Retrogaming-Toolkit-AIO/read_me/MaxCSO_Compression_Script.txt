MaxCSO Compression Script est un outil avec interface graphique pour compresser des fichiers ISO en CSO ou ZSO. Il utilise maxcso.exe pour optimiser la compression.

Fonctionnalités
Compression ISO vers CSO/ZSO : Convertit des fichiers ISO en formats compressés.

Utilisation optimisée des cœurs CPU : Utilise tous les cœurs disponibles.

Téléchargement automatique de maxcso.exe : Télécharge l'outil si absent.

Utilisation
Lancez le script.

Sélectionnez les fichiers ISO à compresser.

Choisissez les options de compression.

Lancez la compression.
