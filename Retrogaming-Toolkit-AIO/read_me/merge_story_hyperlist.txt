Merge Story Hyperlist est un script Python qui intègre des fichiers texte contenant des histoires dans un fichier XML de type hyperlist.

Fonctionnalités
Intégration des histoires : Ajoute ou met à jour la balise <story> dans le fichier XML.

Gestion des erreurs d'encodage : Lit les fichiers texte avec plusieurs encodages.

Sauvegarde des modifications : Enregistre les modifications dans un nouveau fichier XML.

Utilisation
Exécutez le script.

Les histoires sont intégrées dans le fichier XML.
