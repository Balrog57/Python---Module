Game Batch Creator est un outil pour créer des fichiers batch pour lancer des jeux (normaux, Steam, Epic Games).

Fonctionnalités
Création de batch pour jeux normaux, Steam et Epic Games : Génère des fichiers batch pour exécuter des jeux.

Utilisation
Lancez le programme.

Remplissez les champs pour le type de jeu souhaité.

Générez le fichier batch.
