Assisted Gamelist Creator est un outil pour créer et mettre à jour des listes de jeux au format XML. Il permet d'enrichir les informations manquantes via le presse-papiers.

Fonctionnalités
Gestion des fichiers XML : Charge ou crée des fichiers XML.

Enrichissement via le presse-papiers : Intègre les données collées dans le presse-papiers.

Fusion des données : Fusionne les données enrichies avec la liste existante.

Utilisation
Lancez le script.

Chargez la liste de jeux.

Enrichissez les données via le presse-papiers.

Sauvegardez la liste mise à jour.
