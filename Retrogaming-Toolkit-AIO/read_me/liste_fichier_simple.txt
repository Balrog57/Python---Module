Liste Fichier Simple est un script Python qui parcourt un répertoire et ses sous-répertoires pour lister tous les fichiers. La liste est enregistrée dans un fichier texte nommé Liste.txt.

Fonctionnalités
Liste des fichiers : Parcourt récursivement le répertoire spécifié.

Exclusion automatique : Ignore le script en cours et le fichier de sortie.

Enregistrement dans un fichier texte : Sauvegarde la liste dans Liste.txt.

Utilisation
Exécutez le script.

La liste des fichiers est générée dans Liste.txt.
