Folder Name to TXT est un outil pour créer des fichiers texte vides portant le même nom que les fichiers d'une extension spécifiée.

Fonctionnalités
Création de fichiers texte : Génère un fichier texte vide pour chaque fichier ayant l'extension spécifiée.

Utilisation
Exécutez le script.

Entrez l'extension des fichiers.

Les fichiers texte sont créés.
