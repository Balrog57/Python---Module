Media Orphan Detector est un script Python qui identifie et déplace les fichiers multimédias orphelins (sans fichier ROM correspondant) dans un dossier orphan.

Fonctionnalités
Détection des fichiers multimédias orphelins : Vérifie les fichiers multimédias dans medium_artwork.

Déplacement des fichiers orphelins : Les fichiers sans ROM correspondante sont déplacés dans orphan.

Ignorer les fichiers "default" : Les fichiers multimédias nommés "default" sont ignorés.

Utilisation
Exécutez le script.

Les fichiers multimédias orphelins sont déplacés dans orphan.
