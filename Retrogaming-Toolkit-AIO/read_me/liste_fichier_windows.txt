Liste Fichier Windows est un script Python qui liste tous les fichiers et répertoires du répertoire courant et enregistre la liste dans un fichier texte nommé Liste.txt.

Fonctionnalités
Liste des fichiers et répertoires : Parcourt le répertoire courant.

Enregistrement dans un fichier texte : Sauvegarde la liste dans Liste.txt.

Utilisation
Exécutez le script.

La liste des fichiers et répertoires est générée dans Liste.txt.
